# Urban Houses Marketplace

Premium real estate marketplace built with React, Vite, Tailwind CSS, React Router, React Query, and Framer Motion.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Environment

Set these in `.env` when connecting real APIs:

```bash
VITE_API_BASE_URL=https://your-api.example.com
VITE_USE_MOCK_API=true
```
